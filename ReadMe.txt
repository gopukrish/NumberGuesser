Steps to run
------------
1. Run the maven pom.xml which is in top if project folder using 'mvn clean install' command from cli
2. All source code will be compiled and unit test cases will be run.
3. Once build successful, take the generated GuessNumber-0.0.1-SNAPSHOT.jar and place it in any directory
4. Go to jar directory and run' java -jar GuessNumber-0.0.1-SNAPSHOT.jar' command from cli
